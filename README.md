# InterviewAce AI

An AI-powered interview preparation platform that generates recruiter-ready answers, key discussion points, and follow-up questions using Gemini AI.

## Features

- Generate professional interview answers
- Get key interview points
- View common mistakes candidates make
- Receive possible follow-up questions
- Modern responsive UI
- Real-time AI responses using Gemini API

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### AI
- Google Gemini API

## How It Works

User enters an interview question

↓

Frontend sends request to backend

↓

Backend calls Gemini AI

↓

AI generates a structured recruiter-focused answer

↓

Response is displayed in a modern UI
